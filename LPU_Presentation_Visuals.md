## 图表3：内存访问模式

### GPU内存访问（层次化）
```
计算请求 → L1缓存 → 命中？ → 是 → 快速返回
                    ↓否
                 L2缓存 → 命中？ → 是 → 中等延迟
                        ↓否
                     HBM → 高延迟返回
```

### LPU内存访问（直接）
```
计算请求 → 统一SRAM → 固定延迟返回
```
